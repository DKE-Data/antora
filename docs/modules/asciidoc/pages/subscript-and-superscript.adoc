== Subscript syntax

Text is subscripted when enclosed in a set of tildes (`{tilde}`).

.Subscript
[source,asciidoc]
----
The chemical formula for water is H~2~O.
----

.Result
====
The chemical formula for water is H~2~O.
====

== Superscript syntax

Text is superscripted when enclosed in a set of carets (`{caret}`).

.Superscript
[source,asciidoc]
----
What is the answer to E=mc^2^?
----

.Result
====
What is the answer to E=mc^2^?
====
